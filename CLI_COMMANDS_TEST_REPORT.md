# 商家后台助手 - CLI 快捷指令测试报告

## 测试工具

测试脚本: `test_cli_commands.py`

使用方法:
```bash
python test_cli_commands.py [command]

Commands:
    help        - 显示帮助卡片
    accounts    - 显示账号列表
    status      - 显示任务状态
    history     - 显示历史记录
    attachments - 显示附件列表
    stores      - 显示门店管理
    all         - 测试所有命令
```

## 支持的快捷指令

### 1. 帮助类

| 指令 | 响应类型 | 说明 |
|------|----------|------|
| `帮助` | 帮助卡片 | 显示所有支持的命令 |
| `help` | 帮助卡片 | 英文版帮助 |
| `?` | 帮助卡片 | 简短帮助 |
| `怎么用` | 帮助卡片 | 使用帮助 |
| `使用说明` | 帮助卡片 | 使用帮助 |

### 2. 账号类

| 指令 | 响应类型 | 说明 |
|------|----------|------|
| `账号列表` | 账号卡片 | 显示所有账号及状态 |
| `账号` | 账号卡片 | 同上 |
| `我的账号` | 账号卡片 | 同上 |
| `账户列表` | 账号卡片 | 同上 |
| `登录状态` | 账号卡片 | 同上 |

### 3. 状态类

| 指令 | 响应类型 | 说明 |
|------|----------|------|
| `状态` | 文本消息 | 显示当前任务状态 |
| `任务` | 文本消息 | 同上 |
| `任务状态` | 文本消息 | 同上 |
| `运行中` | 文本消息 | 同上 |

### 4. 历史类

| 指令 | 响应类型 | 说明 |
|------|----------|------|
| `历史` | 文本消息 | 显示历史记录提示 |
| `历史记录` | 文本消息 | 同上 |

### 5. 附件类

| 指令 | 响应类型 | 说明 |
|------|----------|------|
| `附件` | 附件卡片 | 显示最近附件 |
| `最近图片` | 附件卡片 | 同上 |
| `文件列表` | 附件卡片 | 同上 |

### 6. 门店类

| 指令 | 响应类型 | 说明 |
|------|----------|------|
| `门店` | 文本消息 | 显示门店信息 |
| `店铺管理` | 文本消息 | 同上 |
| `查看门店` | 文本消息 | 同上 |
| `门店信息` | 文本消息 | 同上 |

### 7. 登录类

| 指令格式 | 响应类型 | 说明 |
|----------|----------|------|
| `登录 <平台> <店铺名>` | 文本消息 | 打开浏览器登录 |
| `打开 <平台> <店铺名>` | 文本消息 | 同上 |
| `进入 <平台> <店铺名>` | 文本消息 | 同上 |

支持的平台:
- `美团` / `美团外卖`
- `饿了么`
- `抖音` / `抖音来客`

示例:
- `登录 美团 江湖饭焗`
- `登录 饿了么 海底捞`
- `登录 抖音来客 瑞幸咖啡`

## 任务指令

非快捷指令的消息会被识别为任务指令，自动提交到任务队列：

示例:
- `美团江湖饭焗修改商家电话为13888888888`
- `把美团 江湖饭焗 营业时间改为 09:00-22:00`
- `把美团 江湖饭焗 门店照片换成刚上传的图片`
